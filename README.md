Connectify 🌐

A full-stack social media web application built using the MERN Stack where users can register, login, create posts, like posts, comment on posts, edit/delete their own posts, and interact with other users securely using JWT Authentication.

🚀 Features
🔐 Authentication & Authorization
User Registration
User Login
JWT Access Token
Refresh Token System
Protected Routes
Secure Authentication Middleware
Auto Login using Refresh Token
Logout Functionality

📝 Post Features
Create Post
Edit Post
Delete Post
View Feed
Like / Unlike Posts
Comment on Posts
Real-time UI Update without Refresh

👥 User Features
Follow / Unfollow Users
User Profile Data
Followers & Following System
